# MSFT Equity Research Models

Generated by Model Builder, Equity Research Swarm
Date: 2026-03-08
Ticker: MSFT (Microsoft Corporation)

All models use Python standard library only. Execute with `python3 <model>.py`.

---

## Models

### 1. msft-dcf-model.py — Discounted Cash Flow
Segment-level revenue build DCF with bull/base/bear scenarios. Projects 5-year free cash flow (FY2026-FY2030), calculates WACC from components, derives terminal value via perpetuity growth and exit multiple methods (50/50 blend), bridges enterprise value to equity value per share, and generates WACC vs. TGR sensitivity tables.

**Key outputs:** Implied share price per scenario, probability-weighted price target, sensitivity table, JSON results.

### 2. msft-comps-model.py — Comparable Company Analysis
Compares MSFT against 7 enterprise software/cloud peers (AMZN, GOOGL, ORCL, CRM, SAP, ADBE, NOW) on EV/EBITDA, P/E, EV/Revenue, and PEG multiples. Calculates statistical summaries, z-scores vs. comp set, implied valuations at comp medians, outlier detection, and composite value+growth ranking.

**Key outputs:** Multiples table, z-scores, implied valuation range, composite ranking, JSON results.

### 3. msft-risk-model.py — Risk and Stress Test
Calculates historical volatility (3-year monthly data), parametric and historical VaR (95%/99%), runs 1,000-path Monte Carlo simulation (12-month horizon), analyzes maximum drawdowns, builds correlation matrix (S&P 500, NASDAQ), computes risk-adjusted return metrics (Sharpe, Sortino, Kelly), and runs 7 stress test scenarios.

**Key outputs:** Volatility metrics, VaR estimates, Monte Carlo distribution, drawdown analysis, stress test results, JSON results.

### 4. msft-credit-model.py — Credit and Balance Sheet
Comprehensive credit analysis covering leverage ratios (Net Debt/EBITDA, Debt/Equity, Interest Coverage), debt maturity profile and refinancing risk, FCF waterfall after all obligations (CapEx, interest, maturities, dividends, buybacks), CapEx sustainability analysis with normalization timeline, Altman Z-Score calculation, off-balance-sheet obligations, and stress testing across 7 revenue growth scenarios (5% to 20%).

**Key outputs:** Leverage/coverage ratios, FCF waterfall, maturity analysis, Altman Z-Score, credit stress test, implied ratings, JSON results.

### 5. msft-scenario-model.py — Scenario Analysis and Kelly Criterion
Defines bull/base/bear price target scenarios with probabilities, calculates probability-weighted expected value, computes Kelly criterion (full/half/quarter) for position sizing, performs breakeven bear probability analysis, and generates structured JSON for chart generation by `tools/report-generator.py`.

**Key outputs:** Expected value price, Kelly fractions, breakeven analysis, risk/reward profile, chart-ready JSON (saved to both `output/models/` and `output/data/MSFT-scenarios.json`).
